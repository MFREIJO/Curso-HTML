normalize.css

El normalize.css nos servirá para realizar ajustes de formatos y posiciones de los elementos en todos los navegadores (normaliza las diferencias entre navegadores).

Podemos ver el código desde github

Para instalarlo lo podemos hacer desde iTerm a través del comando:

npm install normalize.css

Lo que hace es instalar en nuestro proyecto esta estructura

Debemos tener el archivo pagckage.json en el proyecto (importe para poder mantener la estructura actualizada). Para instalarlo ponemos en iTerm:

npm init

Debemos mantener esta estrutura actualizada. Esto lo conseguimos haciendo

npm update

faveicon.ico

Utiliza esta página para generar el icono.

Pasos:

  • Crear una imagen en sketch 1x en PNG de 80x80 (por poner un ejemplo pero sirve cualquier tamaño).
  • Pulsar en la página web el botón Select your Favicon picture.
  • Pulsar en la página web el botón Select your Favicon picture.
  • Nos da un mensaje sobre el tamaño de la imagen pero pulsamos en Continue with this picture.
  • Al final de la página pulsamos Generate your Favicons and HTML code.
  • Se descarga un fichero comprimido y dentro (entre otras imágenes) está el favicon.ico que debemos situar en la carpeta img
